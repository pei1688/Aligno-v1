## 仿Trello應用練習

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Aigno -仿Trello
---------------------
框架:React &Next JS
資料庫:Neon & Prisma
使用hello-pangea/dnd來進行托拽
狀態管理:Zustand管理modal開關狀態
使用React Query與Server action來獲取數據資料
第三方金流串接:Stripe
路由模組化架構提高可維護性
